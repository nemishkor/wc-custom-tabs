Add custom tabs to each WooCommerce product.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-custom-tabs` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Open some woocommerce product
1. After content panels 'Attributes', 'Advanced' you will see new panel 'Custom content tabs'
